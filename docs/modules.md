# Built-in Modules

Spectator provides built-in cybersecurity modules.

## Recon

* Recon(target)
* PortScan(target, start, end)
* DNSLookup(target)

## OSINT

* WHOIs(domain)
* GeoIP(ip)
* SubdomainEnum(domain)

## Web

* HTTPProbe(url)
* Headers(url)
* SSLInfo(domain)

## Exploitation

* SQLiTest(url)
* CMDInject(url)
* PathTraversal(url)

## Payloads

* PayloadGen(type)
* payloadList(type)

## Crypto

* HashIdentify(hash)
* Crack(hash)
